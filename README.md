# 🏋️ Fitness Dashboard (Power BI)

## 📌 Overview

This project is an interactive **Fitness Dashboard** built using Power BI to analyze business performance and client activity in a fitness center.

It provides insights into revenue, expenses, memberships, and client engagement.

---

## 📊 Features

* Revenue, Expenses & Profit Analysis
* Client & Trainer Overview
* Membership Distribution (Platinum, Gold, Silver)
* Monthly Member Trends
* Client Membership Status Tracking

---

## 🎯 Key Highlights

* Clean and modern dashboard design
* Interactive visuals for better insights
* Data-driven decision making
* User-friendly layout

---

## 🛠️ Tools Used

* Power BI
* DAX (Data Analysis Expressions)
* Excel (for dataset)
